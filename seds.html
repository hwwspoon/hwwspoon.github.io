Stellar SEDs for MICE

There are currently five sets of MICE input stellar SEDs available. You may wish to install all of them.

Kurucz SEDs

These are the Kurucz stellar models of 1994. For more information check Kurucz' personal homepage.
Unzip and un-tar the Kurucz SED package in a newly created kurucz subdirectory of directory seds. Change the path in file mice_file_table.pro accordingly.

COSTAR SEDs

These are the stellar models of Schaerer & de Koter (1996) [ Schaerer et al. (1996a) | Schaerer et al. (1996b) ].
Unzip and un-tar the COSTAR SED package in a newly created schaerer subdirectory of directory seds. Change the path in file mice_file_table.pro accordingly.

Pauldrach-1997 SEDs

These are the hot star (25,000-70,000 K) models of Pauldrach, computed in 1997-1998 [Pauldrach et al. (1998)]. Unzip and un-tar the Pauldrach-1997 SED package in a newly created pauldrach1997 subdirectory of directory seds. Change the path in file mice_file_table.pro accordingly.

Pauldrach-2000 SEDs

These are the hot star (25,000-55,000 K) models of Pauldrach, computed in 1999-2000 [Pauldrach et al. (2001)]. Unzip and un-tar the Pauldrach-2000 SED package in a newly created pauldrach2000 subdirectory of directory seds. Change the path in file mice_file_table.pro accordingly.

Hybrid SEDs (=Pauldrach-1997 & Kurucz)

This grid combines the cool stellar models of Kurucz (<19,000K) with the hot stellar models of Pauldrach (>25,000K) computed in 1997-1998. The models for the range 19,500-24500K have been interpolated from the 19,000K Kurucz models and the 25,000K Pauldrach models. Details as to the interpolation method can be found here.
Unzip and un-tar the Hybrid SED package in a newly created hybrid subdirectory of directory seds. Change the path in file mice_file_table.pro accordingly.
